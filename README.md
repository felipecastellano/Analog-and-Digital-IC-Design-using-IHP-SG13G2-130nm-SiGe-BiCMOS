# Analog Inverter Design using IHP Open PDK SG13G2 130nm SiGe BiCMOS Technology

## Description

Este proyecto presenta el diseño e implementación de un negador analógico utilizando la tecnología open source IHP Open PDK SG13G2 de 130nm SiGe BiCMOS.

El desarrollo incluye el diseño esquemático, simulación y generación del layout físico del circuito integrado mediante herramientas EDA de código abierto. Además, se realizó la exportación del diseño en formato GDSII para su visualización y verificación física.

La tecnología SG13G2 emplea dispositivos basados en Silicon-Germanium (SiGe), ampliamente utilizados en aplicaciones analógicas, RF y de alta velocidad.

## Tools Used

- KLayout
- xschem
- ngspice
- IHP Open PDK SG13G2

## Notes

El proyecto fue desarrollado con fines académicos y de aprendizaje en diseño de circuitos integrados analógicos usando flujos open source.
